ANIMATED PUZZLE

Rotating blocks



Animated Puzzle is a free and offline photo puzzle maker.
With Animated Puzzle you can create puzzles with rotating and fading picture blocks.

This app is based on the classic game where we move blocks to complete a picture.
To play, first add your own photos to the application.
When all the blocks have stopped in their right position, the player wins.

The Animated Puzzle supports three modes
- static, use only static picture blocks
- rotating, use rotating and static picture blocks
- fading, use fading and static picture blocks
